HTML

HTML stands for Hypertext Markup Language and while it is the structure of most if not all websites it is not a programming language. It is actually a set of characters and symbols that when combined in a HTML document produce the text, images and links on a webpage. When typed out correctly a web browser gathers the data and you will see a bunch of writing and links though it won't be anything too fancy. Remember this is just the first part of the house, the structure.

CSS

I feel like this is where the fun truly begins, and though I like the Markup as it gets me too this point I really like the CSS as I have always loved design. CSS stands for Cascading Stylesheets, and is used to make the site look more appealing to visitors. It allows you to add colors, fonts, and configure your layouts to so that things flow better for the visitor of your website.
